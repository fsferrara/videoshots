Introduzione

E' stato sviluppato un software web-based utilizzando il linguaggio di programmazione Java lato server e il framework JMF (Java Media Framework), per la gestione di un archivio videonews. Ogni videonews e' composta da:
- un filmato
- un'immagine (in formato jpeg)
- una descrizione XML in formato NITF (Standard dell'IPTC)

Per accelerare il processo di installazione, all'interno del CD-ROM fornito con il progetto sono presenti, nella cartella VideoConvertiti, tutti i filmati convertiti in formato ".mov" con codec video H.263 e codec audio WAV_PCM 16 Big-Endian.

Le videonews sono caricate nell'archivio mediante una funzione di upload, che provvede a convertire il file XML in formato NITF in un file XML in formato Mpeg-7. Durante questa operazione sono cercate le videonews correlate "related" con la videonews appena caricata: cio se viene trovata un'altra videonews che ha almeno due caratteristiche in comune, si modificano e descrizioni Mpeg-7 di entrambe le videonews aggiungendo riferimenti incrociati.

Il software sviluppato permette di cercare le videonews utilizzando come criteri di ricerca: autore, categoria (spettacolo, cronaca, estero, politica, ecc), location; l'utente puo' ad esempio cercare tutte le videonews prodotte dal medesimo autore e che siano della stessa categoria

Le descrizioni Mpeg-7 possono essere arricchite dall'utente con informazioni semantiche di shot appartenenti alle videonews. Inoltre, e' possibile effettuare la ricerca nelle descrizioni degli shot definiti, e quindi selezionare solo parti di filmati.

La visualizzazione dei risultati, e di tutte le altre pagine del sito, e' realizzata usando fogli di stile xsl.

Una descrizione piu' approfondita e' reperibile sul CD-ROM consegnato assieme al progetto.
